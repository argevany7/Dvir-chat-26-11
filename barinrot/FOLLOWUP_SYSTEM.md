# מערכת פולואו אפ - אריאל

## סקירה כללית
מערכת פולואו אפ אוטומטית שפועלת לפי דרגות חום ומתאימה הודעות אישיות לכל לקוח.

## לוגיקת הפולואו אפ

### שלב 1 (ניסיונות 1-3): פולואו אפ יומי
- **תדירות**: כל יום
- **שעה**: רנדומלית בין 12:00-19:00
- **הודעות**: הודעות מעודדות וסקרניות

### שלב 2 (ניסיונות 4-5): פולואו אפ כל יומיים
- **תדירות**: כל יומיים
- **שעה**: רנדומלית בין 12:00-19:00
- **הודעות**: הודעות עדינות יותר, פחות לוחצות

### שלב 3 (ניסיון 6+): פולואו אפ כל שלושה ימים
- **תדירות**: כל שלושה ימים
- **שעה**: רנדומלית בין 12:00-19:00
- **הודעות**: הודעות סופיות, מכבדות

## תכונות מיוחדות

### שליחת GIF
- **ההודעה השלישית** תמיד תכיל את הקובץ `followUp.gif`
- שאר ההודעות יהיו הודעות טקסט רגילות

### הודעות מותאמות אישית
- כל הודעה מכילה את שם הלקוח
- הודעות שונות למבוגרים וילדים
- הודעות שונות לפי מספר הניסיון
- **הודעות מותאמות לפי מצב הלקוח:**
  - לקוח שקיבל קישור תשלום אבל לא שילם
  - לקוח שדיבר איתנו אבל לא הגיע למצב של קביעת אימון
- **זיהוי בעיות מהסיכום:**
  - לחץ ועצבים
  - ביטחון עצמי
  - אנרגיות
  - בריונות

### מערכת השהיה חכמה
- המערכת עוקבת אחרי הודעות אחרונות של כל משתמש
- השהיה של 10 שניות בין הודעות למניעת מענה כפול
- השהיה של 3 שניות לפני מענה אם המשתמש שלח הודעה לאחרונה
- מונעת מענה כפול שיכול לחשוף AI
- עובד על כל סוגי ההודעות (רגילות ופולואו אפ)

### מנגנון עצירה
הלקוח יכול לבקש להפסיק את הפולואו אפ באמצעות מילות מפתח:
- "תפסיק לשלוח"
- "תפסיק לכתוב"
- "אל תשלח יותר"
- "די"
- "מספיק"
- ועוד...

## שדות במאגר הנתונים

### טבלת clients
- `followup_enabled` - האם פולואו אפ מופעל (BOOLEAN)
- `followup_attempts` - מספר ניסיונות פולואו אפ (INTEGER)
- `last_followup_date` - תאריך הפולואו אפ האחרון (DATETIME)
- `next_followup_date` - תאריך הפולואו אפ הבא (DATETIME)
- `followup_stage` - השלב הנוכחי (TEXT: daily/every_two_days/every_three_days)

## API Endpoints

### בדיקת סטטוס פולואו אפ
```
GET /api/followup-status/:phone
```

### הפעלת פולואו אפ ידני
```
POST /api/trigger-followup
```

## הפעלה אוטומטית

### טיימר
- המערכת בודקת כל 30 דקות אם יש לקוחות הזקוקים לפולואו אפ
- הפולואו אפ מופעל אוטומטית כשהשיחה מסתיימת

### תנאים לפולואו אפ
- השיחה הסתיימה (`conversation_ended = TRUE`)
- הלקוח לא שילם (`payment_confirmed = FALSE`)
- הפולואו אפ מופעל (`followup_enabled = TRUE`)
- הגיע הזמן לפולואו אפ הבא

## הודעות לדוגמה (מעודכן נובמבר 2025)

### ניסיון 1 - הודעה מותאמת למצב
```
היי [שם], חשבת על האימונים? [הקשר ספציפי למצב הלקוח]
```

### ניסיון 2 - GIF בלבד
```
[GIF בלבד ללא טקסט]
```

### ניסיון 3 - פשוט וקצר
```
[שם], עדיין רלוונטי? 😊
```
או
```
היי [שם], עדיין מעניין? 😊
```

### ניסיון 4 - אישית עם נקודות כאב
```
היי [שם]! 😊

אני זוכר שדיברנו על [נקודת כאב ספציפית]. 
דרך אומנויות הלחימה דביר עוזר לבנות ביטחון עצמי וכוח פנימי.

זו באמת הזדמנות לשנות דברים מקצה לקצה 💪
```

### ניסיון 5 - FOMO
```
[שם], מה דעתך על מה שדיברנו? זו באמת הזדמנות לטפל ב"[נקודת כאב]" ולשנות דברים מקצה לקצה 💪

אני בטוח שזה יכול לעזור, לא כדאי לפספס את זה. מה דעתך?
```

### ניסיון 6+ - שיחה כנה ✨ (מעודכן)
**למבוגרים:**
```
היי [שם], אני יודע שזה לא קל להחליט 😊

אבל רציתי להגיד - האימונים האלה יכולים לשנות לך את הביטחון העצמי, הכושר והאיזון המנטלי. 

מה דעתך שנדבר על זה?
```

**להורים:**
```
היי [שם], אני מבין שיש הרבה דברים לחשוב עליהם 😊

אבל האימונים האלה זו באמת השקעה ב[שם הילד] - לא רק פיזית, גם מנטלית וחברתית.

עדיין מעניין?
```

### הודעות Early Rejection (לקוחות שביטאו אי-עניין) ✨ (חדש)
**לאחר שבועיים מאי-מענה:**
```
היי [שם], רק רציתי לחזור אליך - הרבה אנשים מתלבטים בהתחלה אבל אחרי האימון הראשון מבינים שזה בדיוק מה שהיו צריכים 💪

האימונים זה לא רק כושר - זה שחרור לחץ, בניית ביטחון ושינוי אמיתי באיכות החיים.

מה דעתך לנסות?
```

4. **כיבוי ידני**: הלקוח יכול לבקש להפסיק בכל עת
5. **שמירת היסטוריה**: כל הודעת פולואו אפ נשמרת בהיסטוריית השיחה
