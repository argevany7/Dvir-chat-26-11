# 📱 הוראות שימוש - חסימה גמישה

## 🎯 מה חדש?

עכשיו אפשר לחסום לקוחות בצורה גמישה:
- רק מבוט (לא יענה להודעות, אבל פולואו אפ ימשיך)
- רק מפולואו אפ (לא יקבל פולואו אפ, אבל הבוט יענה)
- גם וגם (חסימה מלאה)

---

## 📛 איך חוסמים? - `/block`

### 1. שלח פקודה:
```
/block 0501234567
```

### 2. הזן שם:
```
משה כהן
```

### 3. בחר סוג חסימה:
```
1 - רק בוט (לא יענה, אבל פולואו אפ ימשיך)
2 - רק פולואו אפ (לא יקבל פולואו אפ, אבל הבוט יענה)
3 - שניהם (חסימה מלאה)
4 - חזרה
```

הקלד `1`, `2`, `3` או `4`

---

## 🔓 איך מסירים חסימה? - `/unblock`

### דרך 1: תפריט
```
/unblock
```
בחר אופציה:
- `1` - הזן מספר טלפון
- `2` - הזן מספר סידורי (מחיפוש קודם)
- `3` - חזרה

### דרך 2: מהיר
אם יש תוצאות חיפוש:
```
/unblock 5
```

### אחרי שמצאת לקוח:
הבוט יציג מה להסיר:
```
1 - הסר חסימת בוט
2 - הסר חסימת פולואו אפ  
3 - הסר הכל
4 - חזרה
```

---

## 🔍 איך מחפשים? - `/check`

```
/check
```

בחר:
- `1` - חיפוש לפי מספר
- `2` - חיפוש לפי שם

התוצאות יראו:
```
1️⃣ משה כהן
   📞 972501234567
   🚫 חסום מבוט וגם מפולואו אפ
   
2️⃣ שרה לוי
   📞 972509999999
   🚫 חסום מפולואו אפ בלבד
```

---

## 💡 דוגמאות מהירות

### חסימה מפולואו אפ בלבד:
```
/block 0509999999
שרה לוי
2
```

### הסרת חסימה מלאה:
```
/check
1
0501234567
3
```

### חסימה מלאה:
```
/block 0527777777
יוסי מציק
3
```

---

## ⚡ טיפים

- בכל שלב: `/cancel` - ביטול
- חזרה לתפריט: `/check`
- עזרה: `/help`

---

## ⚠️ חשוב לדעת

✅ רק מנהלים (דביר ואריאל) יכולים לחסום  
✅ חסימה מבוט ≠ חסימה מפולואו אפ  
✅ אפשר לשנות סוג חסימה בכל עת  
✅ בכל תפריט יש אופציה לחזור
