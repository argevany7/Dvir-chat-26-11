## בעיות אפשריות וטיפול

### 1. שגיאת "OPENAI_API_KEY not found"
**פתרון:** ודא שיש לך קובץ `.env` עם המפתח:
```
OPENAI_API_KEY=sk-...
```

### 2. שגיאת "Cannot find module whatsapp-web.js"
**פתרון:** התקן את החבילות:
```bash
npm install
```

### 3. הסקריפט תקוע על "מאתחל WhatsApp Client"
**פתרון:** 
- סרוק את ה-QR אם מופיע
- או מחק את התיקייה `.wwebjs_auth/test-marketing-messages` ונסה שוב
