## 🐛 פתרון תקלות מהיר

### הבוט לא עונה?
```bash
# 1. בדוק שהשרת רץ
ps aux | grep "node server.js"

# 2. בדוק חיבור ווטסאפ
curl http://localhost:3001/status

# 3. הפעל מחדש
killall node
node server.js
```

### ווטסאפ מתנתק?
```bash
# נקה cache של Chrome
rm -rf /tmp/chrome-user-data

# הפעל מחדש
node server.js
```

### GPT לא עובד?
```bash
# בדוק שיש API Key
cat .env | grep OPENAI_API_KEY

# אם אין - צור אחד:
echo "OPENAI_API_KEY=sk-..." > .env
```
