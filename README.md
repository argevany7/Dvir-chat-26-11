# דביר בסון - מאמן אומנויות לחימה - בוט צ'אט

בוט צ'אט חכם שמדבר כמו דביר בסון, מאמן אומנויות לחימה, ומשתמש בבסיס הידע המלא שלו.

## מה זה כולל?

- 🎯 דף צ'אט יפה ומתקדם עם אנימציות
- 🤖 חיבור ל-ChatGPT API עם זיכרון שיחה
- 📚 בסיס ידע מלא של דביר עם חיפוש חכם
- 🎨 עיצוב רספונסיבי ומודרני
- 🇮🇱 תמיכה מלאה בעברית
- 👨‍🏫 התנהגות אנושית מאוד - הקלדה אנושית, הודעות מקריות, רגשות
- 💬 זיכרון שיחה - דביר זוכר מה דיברתם עליו
- ⚡ תגובות מהירות וחכמות
- 🎭 אישיות מלאה של דביר - סגנון דיבור, סיפורים, אימוג'ים

## מה צריך להתקין?

### 1. Node.js
הורד והתקן Node.js מ: https://nodejs.org/
(גרסה 16 ומעלה)

### 2. OpenAI API Key
- לך לאתר: https://platform.openai.com/
- הירשם/התחבר
- צור API Key חדש
- העתק את הטוקן

## איך להפעיל?

### שלב 1: התקנת התלויות
```bash
npm install
```

### שלב 2: הגדרת הטוקן
צור קובץ `.env` בתיקיית הפרויקט:
```bash
cp env_example.txt .env
```

ערוך את הקובץ `.env` והכנס את הטוקן שלך:
```
OPENAI_API_KEY=sk-your-real-api-key-here
```

### שלב 3: הפעלת השרת
```bash
npm start
```

### שלב 4: פתיחת הדפדפן
לך לכתובת: http://localhost:3000

## קבצים בפרויקט

- `index.html` - דף הצ'אט הראשי
- `server.js` - השרת Node.js
- `package.json` - תלויות הפרויקט
- `dvir_basson_knowledge_base.json` - בסיס הידע של דביר
- `.env` - הגדרות (טוקן API)
- `README.md` - המדריך הזה

## איך זה עובד?

1. המשתמש כותב שאלה בדף הצ'אט
2. השרת מחפש מידע רלוונטי מבסיס הידע של דביר
3. השרת יוצר prompt דינמי עם המידע הרלוונטי
4. ChatGPT מקבל את ה-prompt ומענה כמו דביר
5. התשובה מקבלת "מגע אנושי" - אימוג'ים, תגובות מקריות
6. התשובה מוצגת עם אנימציית הקלדה אנושית
7. השיחה נשמרת בזיכרון למשך הפגישה

## התכונות האנושיות:

- **הקלדה אנושית**: התשובות מוקלדות בקצב משתנה עם הפסקות
- **הודעות מקריות**: דביר שולח הודעות מעודדות מדי פעם
- **זיכרון שיחה**: זוכר את מה שדיברתם עליו
- **רגשות**: משתמש באימוג'ים ובתגובות אנושיות
- **סיפורים אישיים**: מספר על החיים שלו ועל הניסיון
- **סגנון דיבור**: מדבר כמו דביר האמיתי עם המילים שלו

## התאמות אפשריות

### שינוי מודל
בקובץ `server.js`, שורה 58, תוכל לשנות:
```javascript
model: "gpt-4", // או "gpt-3.5-turbo" לזול יותר
```

### שינוי פורט
בקובץ `.env`:
```
PORT=8080
```

### הוספת מידע לבסיס הידע
ערוך את `dvir_basson_knowledge_base.json` והוסף שאלות ותשובות חדשות.

## פתרון בעיות

### שגיאה: "API Key לא תקין"
- ודא שהטוקן ב-`.env` נכון
- ודא שיש לך אשראי ב-OpenAI

### שגיאה: "Port כבר בשימוש"
- שנה את הפורט ב-`.env`
- או עצור את התהליך שכבר רץ על הפורט

### הצ'אט לא עובד
- פתח Developer Tools בדפדפן (F12)
- בדוק את הלשונית Console לשגיאות
- ודא שהשרת רץ (צריך לראות הודעות בקונסול)
